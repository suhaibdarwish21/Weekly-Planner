#  Weekly Planner App

A dynamic JavaScript app to manage weekly shopping items with priority sorting

##  Features
- Sort items by importance  
- Change item colors dynamically  
- Simple and responsive layout  

##  Technologies
- HTML  
- CSS  
- JavaScript
